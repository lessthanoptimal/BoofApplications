This jar contains example and demonstration code for BoofCV. Example code is intended to show how to do specific
tasks in an easy to understand way. Demonstration code is more focused on visualization and more difficult to
understand.

Requirements:

Java 1.8 or later
Java 11 recommended

Convenient scripts have been provided for launching the jar that should work in Windows, Mac OS X, and Linux.
Alternatively you can just launch the jar directory with the command below

java -jar demonstrations.jar
